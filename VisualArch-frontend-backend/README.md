# 🌌 Flactuation - quantumv1.0

> **Empowering architectural vision with AI-Native precision.**

**Flactuation** is a cutting-edge platform designed to bridge the gap between architectural blueprinting and ready-to-use codebase generation. Using a multi-agent "Quantum Orchestrator," it allows developers to design complex systems visually on a canvas and generate a full-stack implementation (React/Node.js/MongoDB) in a single shot.

---

## 🚀 Key Features

- **Quantum Orchestrator**: A multi-agent AI framework with built-in fallbacks (Gemini, Groq, OpenRouter).
- **Infinite Canvas**: Interactive architectural modeling with automatic node layout and relationship mapping.
- **Atomic Design System**: Automated UI/UX wireframing that respects architectural constraints.
- **Stateful Memory**: AI agents that "remember" your architectural decisions across iterations.
- **One-Shot Generation**: From high-level prompt to running code in under 60 seconds.

## ⚙️ Quick Start

### 1. Requirements
- Node.js 20+
- MongoDB instance (Local or Atlas)
- AI API Keys (OpenRouter, Groq, or Gemini)

### 2. Installation

```bash
# Clone the repository
git clone https://github.com/your-repo/flactuation.git

# Setup Environment
cp ./visualflow-backend/.env.example ./visualflow-backend/.env
```

### 3. Run Development

**Backend:**
```bash
cd visualflow-backend
npm install
npm run dev
```

**Frontend:**
```bash
cd Flactuation-frontend
npm install
npm run dev
```
